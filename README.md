# Bootstrapping in Applications
Contains work written on the useful connections between statistical theory and practice.

This document was written to provide a reasonably comprehensive introduction to the bootstrap method of estimating standard errors or confidence intervals, with a particular focus on connecting some of the theory with applications. It may be of use to a trained statistician who has not been formally introduced to bootstrap methodology or theory, but who is interested in understanding how it can be used to create the several different methods of constructing bootstrap estimators, and what the important assumptions are behind their construction. After a close reading of this document, the reader should know the differences between different bootstrap estimators, how to write practical code to implement them, and their strengths and weaknesses, including in which situations one would want to use one type versus another.

This document was originally written as a supplement to a several hour long presentation on bootstrap methods.
